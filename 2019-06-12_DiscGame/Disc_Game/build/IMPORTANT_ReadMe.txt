For the game to work, make sure to build (ie. double click "build.bat"), because that is where and how the "sources" file is established.
This file is crucial to this game. Building creates the new file paths that are different if they are allocated in different folders.
After running "buld.bat", double click the "run.bat" file.

Additionally, if there is an error with running the game, make sure the entire path of the file has no spaces in there because if there are spaces in the path, the game will not work.
For exapmle: "F:\Final_Project\DiscGame_VincentZhang\Disc_Game\build" has no spaces in there at all so the game will work.
If the path was "F:\Final Project\DiscGame VincentZhang\Disc_Game\build" it will not work.
